# SalesXP

Application gamifiée d'entraînement à la vente complexe pour les commerciaux en ESN.

## Concept

Transforme l'apprentissage commercial en jeu : scénarios réalistes, choix stratégiques,
feedback immédiat, récompenses et progression de niveau. Passe de vendeur simple à
chasseur de deals complexes.

## Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Supabase (PostgreSQL + Auth)
- **Déploiement**: Vercel

## Démarrage

```bash
npm install
npm run dev
```
